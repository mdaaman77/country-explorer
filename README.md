# 🌍 Country Explorer (JS + React)

A modern, high-performance country explorer built with **Vanilla JavaScript** and **React + Tailwind CSS**.  
Designed with scalability, clean architecture, and premium UI in mind.

---

## ✨ Live Features

- 🔍 Search countries by **name, code, currency, region**
- 🎯 Population filtering with **interactive range slider**
- 📊 Sorting by **name & population**
- 🌐 Country detail page with extended insights
- ⚡ Optimized performance using **debouncing & memoization**
- 🎨 Modern UI with **glassmorphism + gradient themes**
- 📱 Fully responsive design

---

## 🧱 Tech Stack

### JS Version
- HTML5
- CSS3 (custom modern UI)
- Vanilla JavaScript (modular architecture)

### React Version
- React (Vite)
- Tailwind CSS
- React Router
- Custom Hooks
- REST Countries API
